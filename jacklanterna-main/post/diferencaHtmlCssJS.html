Como funciona a HTML e as CSS?

Começamos por dizer que HTML e CSS não são linguagens de programação, e sim de marcação e estilização;

Por que não posso dizer que programo em HTML?

Por vários motivos, como por exemplo, HTML não tem estrutura que as linguagens de programação tem, como variáveis, condições, repetições, vetores, objetos, e acesso a banco de dados e outras funcionalidades como o JavaScript, Java, Python, PHP, Ruby C#, C++ e outras linguagens tem.

A linguagem HTML é uma linguagem focada em conteúdo, como textos, imagens, vídeos, tabelas e tudo que for focado em conteúdo, mais é claro que não adianta ter um site com muito conteúdo, sem uma organização e um design bonito e atrativo, e é nesse cenário que entra as CSS. Cores, sombras, tamanhos e posicionamentos, são apenas algumas das funções do CSS.

Um pouco de JavaScript

Em sequência vem o javaScript, para interações com menus, animações, pop up e uma infinidade de outras funcionalidades.

O texto alternativo?

Estilo CSS

CSS é o design, as mudanças de cor,os trechos que devem ser destacados, tamanho das imagens, divisão de conteúdos e diversas outras coisas. O estilo CSS geralmente está dentro de uma tag HTML, mas ele utiliza seletores.

JavaScript

JavaScript é uma linguagem de programação criada para ser parte dos navegadores web, para que scripts possam ser executados do lado do cliente e interajam com o usuário sem a necessidade deste script passar pelo servidor, controlando o navegador, realizando comunicação assíncrona e alterando o conteúdo do documento exibido.

Essas três linguagens são a base de conhecimento de um desenvolvedor Front-End, que é o profissional que vai trabalhar com elas de forma tão uniforme, que vai garantir que o site criado seja super atrativo e de fácil leitura
