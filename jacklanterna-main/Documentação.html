Introdução HTML

Novembro 15, 202 Por RoniWellington

A linguagem HTML é uma linguagem focada em conteúdo, como textos, imagens, vídeos, tabelas e tudo que for focado em conteúdo.

HTML significa HyperText Markup Language(Linguagem de Marcação de HiperTexto)

Introdução CSS

CSS é a linguagem que usamos para estilizar uma página da Web. Descreve como os elementos HTML devem ser posicionados na tela.

CSS significa Cascading Style Sheets( Folhas de Estilo em Cascata)

JavaScript

JavaScript é a linguagem de programação mais popular do mundo, muito utilizada para desenvolvimento web, com uma curva fácil de aprendizagem.

Lógica de programação

Algoritmos

Algoritmos são passos a serem seguidos com a intenção de atingir um objetivo, que pode ser desde atravessar a rua, fazer café até critérios para seleção de alunos.

Algoritmo AtravessarRua

  1. Olhar para a direita
  2. Olhar para a esquerda
  3. Se estiver vindo carro
  4. Não Atravesse
  5. senão
  6. Atravesse
  7. Fim

Lógica de programação

Lógica de programação é onde aplicamos todos os conceitos de algoritmos em uma linguagem de programação.

Git e GitHub

Plataforma de compartilhamento de códigos que realiza o controle de versão por meio do Git. O GitHub permite que vários desenvolvedores colaborem em conjunto em um mesmo projeto, fazendo modificações e registrem o seu progresso.

Bootstrap

Bootstrap é um framework web com código-fonte aberto para desenvolvimento de componentes de interface e front-end para sites e aplicações web, usando HTML, CSS e JavaScript.

React

O React é uma biblioteca front-end JavaScript de código aberto com foco em criar interfaces de usuário em páginas web.

NodeJS

Node.js é um software de código aberto, multiplataforma, baseado no interpretador V8 do Google e que permite a execução de códigos JavaScript fora de um navegador web.

Linux

Linux é um termo popularmente empregado para se referir a sistemas operativos ou sistemas operacionais que utilizam o Kernel Linux. O núcleo foi desenvolvido pelo programador finlandês Linus Torvalds, inspirado no sistema Minix.

Arquitetura de Computadores

A arquitetura de computadores é a forma como os diversos componentes de um computador são organizados, determina aspectos relacionados à qualidade, ao desempenho e à aplicação para a qual o dispositivo vai ser orientado.

Python

Python é uma linguagem de programação de alto nível, interpretada de script, imperativa, orientada a objetos, funcional, de tipagem dinâmica e forte. Foi lançada por Guido van Rossum em 1991.

Java

Java é uma linguagem de programação orientada a objetos desenvolvida na década de 90 por uma equipe de programadores chefiada por James Gosling, na empresa Sun Microsystems, que em 2008 foi adquirido pela empresa Oracle Corporation.
