wikipédia Elon-Musk

“Se você não consegue explicar algo de forma simples então você não entendeu o assunto o suficiente”

Técnica Feynman

A técnica foi criada pelo físico Richard Feynman, vencedor do prêmio nobel de 1965, venerado pela habilidade de explicar assuntos extremamente complexos como por exemplo a física quântica para praticamente qualquer pessoa de forma simples e interessante.

Princípio básico ===> verdade {
O que eu realmente sei?
} o que eu com certeza não sei
Técnica passo-a-passo:

Passo Nº 1:{
Escolha um assunto que você queira entender e comece a estudá-lo.
} aqui a dica é a seguinte:{
Busque o máximo de conteúdo disponível seja vídeos, livros, artigos, sites ou qualquer fonte que te interesse.
} E ao estudar {
Escreva em tópicos tudo que você está aprendendo sobre o assunto e, de preferência a mão , que é uma das formas mais eficazes de reter conteúdo
}

Passo Nº 2: {
Finja ensinar o conteúdo aprendido para alguém .
} A ideia aqui é {
que você organize as ideias anotadas em tópicos do Passo anterior e ordenar essas ideias em uma ordem lógica de começo meio e fim ,}
{ como se você estivesse ensinando para alguém, isso ajuda e reter a informação, sempre use termos simples, evitando ao máximo o uso de jargões técnicos para facilitar o entendimento de quem ouve }

Passo Nº 3: {
Identifique suas falhas na explicação, então durante a explicação você vai naturalmente identificar as partes que você se sentiu inseguro, quando você foi complexo demais, simplesmente faça uma marcação dos pontos a certifique-se de que podem ser melhorados
}

Passo Nº 4: {
Corrigir o conteúdo aprofundando o conhecimento e simplificando a explicação } então se for uma falha de conteúdo {
Volte para o passo 1{
estude mais a fundo e reforce as partes de conhecimentos que precisam ser aprofundados. }falha de complexidade excessiva{
volte estude e simplifique ao máximo resumindo conteúdos longos e sempre usando analogias para explicar termos complexos
}

PassoNº 5 : {
Refaça todo o processo quantas vezes for necessário, até você sentir realmente confiante na sua explicação, que vai mostrar que você realmente aprendeu o conteúdo }

Partes mais fundamentais da técnica Feynman
Uso de analogias (Analogias visuais ) => {
Analogias visuais geram conexão com nível de entendimento de quem está te ouvindo, ( forma mais simples de explicar um fenômeno desconhecido e complexo através de comparações com imagens e diagramas e conceitos visuais que são facilmente compreendidos pelo ouvinte
)}.
