Tailwind CSS (framework front-end)

nov 22, 2022 por RoniWellington

Esta postagem é apenas um resumo, o link para o => Tailwindcss



Tailwind CSS é uma estrutura CSS de código aberto . A principal característica desta biblioteca é que, ao contrário de outros frameworks CSS como o Bootstrap, ela não fornece uma série de classes predefinidas para elementos como botões ou tabelas. Em vez disso, ele cria uma lista de classes CSS "utilitárias" que podem ser usadas para estilizar cada elemento misturando e combinando.

Por exemplo, em outros sistemas tradicionais, haveria uma classe message-warningque aplicaria fundo amarelo e texto em negrito. Para obter esse resultado no Tailwind, seria necessário aplicar um conjunto de classes criadas pela biblioteca: bg-yellow-200e font-bold.

Características

Devido à diferença de conceitos básicos em relação a outros frameworks CSS tradicionais como o Bootstrap , é importante conhecer a filosofia a partir da qual o Tailwind foi criado, bem como sua utilização básica.

Classes utilitárias

O conceito de utilidade em primeiro lugar refere-se ao principal diferencial do Tailwind. [5] Em vez de criar classes em torno de componentes (botão, painel, menu, caixa de texto...), as classes são construídas em torno de um elemento de estilo específico (cor amarela, fonte em negrito, texto muito grande, elemento central...). Cada uma dessas classes é chamada de classes utilitárias .
